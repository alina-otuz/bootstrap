Sport World

A website exploring global sports trends and achievements

Project Overview

Sport World is an educational and informational website that brings together stories, history, and celebration of sports from around the world. It showcases sports such as skiing, boxing, tennis, football, athletics and highlights major themes like teamwork, records, and global unity.

Structure

The site is organized with:

Navigation bar with sport categories and global industry highlights

Content blocks each with images and “Learn More” links

Forms for user interaction (registration, ratings, contact)

Multi-language support for accessibility

Technologies & Setup

HTML5 / CSS3 for structure and styling

Possibly JavaScript for form behaviour

Hosted via GitHub Pages (https://alina-otuz.github.io/sports-world/
)

To deploy/update: commit changes to the repository and push — GitHub Pages will serve the updated site

How to Run Locally

Clone the repository to your local machine.

Open the index.html in your browser to view the site.

(Optional) Use a local web server if needed (e.g., live-server extension or python -m http.server).

Make edits to content, images, languages or styling as required.

Commit and push changes; GitHub Pages will reflect them.

Contribution Guidelines

Feel free to fork the repo and submit pull requests.

New sections or sports can be added — define the sport, add an image, descriptive text, and a “Learn More” link.

For translation support: add language-specific files or adapt existing template.

Please keep code and structure consistent with existing sections for uniformity.

✅ Goal & Vision

The vision of Sport World is to offer a global sports hub that is accessible to a wide audience (English, Russian, Kazakh speakers), to inform and inspire people about the power of sport in history and modern society.
